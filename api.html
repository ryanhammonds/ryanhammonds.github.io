    Cycle-by-cycle¶

    Features Functions¶

    compute_features(x, Fs, f_range[, …])

    Segment a recording into individual cycles and compute features for each cycle

    Segmentation Functions¶

    find_extrema(x, Fs, f_range[, boundary, …])

    Identify peaks and troughs in a time series.

    find_zerox(x, Ps, Ts)

    Find zerocrossings within each cycle after peaks and troughs are identified.

    Waveform Phase Estimation Functions¶

    extrema_interpolated_phase(x, Ps, Ts[, …])

    Use peaks (phase 0) and troughs (phase pi/-pi) to estimate instantaneous phase.

    Burst Detection Functions¶

    detect_bursts_cycles(df, x[, …])

    Compute consistency between cycles and determine which are truly oscillating.

    detect_bursts_df_amp(df, x, Fs, f_range[, …])

    Determine which cycles in a signal are part of an oscillatory burst using an amplitude thresholding approach

    twothresh_amp(x, Fs, f_range, amp_threshes)

    Detect periods of oscillatory bursting in a neural signal by using two amplitude thresholds.

    Plotting Function¶

    plot_burst_detect_params(x, Fs, df_shape, …)

    Create a plot to study how the cycle-by-cycle burst detection algorithm determine bursting periods of a signal.
